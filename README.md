# crime_story_generation
Welcome to Group Hot Dogs' story generation project! Please follow the instructions below to run our code and generate your own story.

1. If you do not have the OpenAI library installed on Python, please create a virtual environment and run "pip install -r requirements.txt" in your terminal.
2. In the terminal, run "python story_generation.py".
3. Wait a while (typically 10-20 minutes) as your story generates!
